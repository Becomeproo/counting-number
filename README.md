## ℹ️앱 설명

화면 가운데에 숫자를 보여주고 + 버튼을 누르면 숫자가 1씩 올라가고, 초기화 버튼을 누르면 숫자가 초기화되어 0이 되는 간단한 앱.

<br>
<br>

## ✅구현 기능

* '+' 버튼 클릭 시, 숫자를 1씩 올리기
* 초기화 버튼을 클릭 시. 숫자를 0으로 변경하기
